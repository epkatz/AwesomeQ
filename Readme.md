AwesomeQ
===

A simple queuing system based on Amazon's Simple Queuing System

AwesomeQ allows multiple producers to write and read from it.

### Running Server
npm install
npm run-script run

### Running cron
cd cron
npm install
npm run-script run

### Running Producer 
cd producer
npm install
npm run-script run

### Running Consumer
cd consumer
npm install
npm run-script run
